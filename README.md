## Паттерны
### Singleton
[Singleton](src/main/kotlin/patterns/Singleton.kt) - порождающий паттерн. Как правило применяется для чтения настроек 
из файла конфигурации, или же в виде класса для подкоючения к базе данных.

### Builder
[Builder (Строитель)](https://github.com/klesarev/kotlin-learn/blob/master/src/main/kotlin/patterns/Builder.kt) - это порождающий паттерн проектирования, который позволяет создавать сложные объекты пошагово.
Применить в автотестах можно в качестве файла для создания конфигурации.

### Observer
[Паттерн Observer](src/main/kotlin/patterns/Observer.kt), или Наблюдатель... 

### State
[State](src/main/kotlin/patterns/State.kt), или Состояние - это поведенческий паттерн проектирования, который позволяет объектам менять поведение 
в зависимости от своего состояния. Извне создаётся впечатление, что изменился класс объекта.

## Алгоритмы
### Числа Фибоначчи
[Числа Фибоначчи](src/main/kotlin/algorithms/Fibonacci.kt) (строка Фибоначчи) — числовая последовательность, первые два числа которой являются 0 и 1, а каждое 
последующее за ними число является суммой двух предыдущих.

### Факториал числа
[Факториалом числа](src/main/kotlin/algorithms/Factorial.kt) называют произведение всех натуральных чисел до него включительно. Например, факториал числа 5 равен 
произведению 1 * 2 * 3 * 4 * 5 = 120.

### FizzBuzz
[FizzBuzz](src/main/kotlin/algorithms/FizzBuzz.kt) - часто встречающаяся задача на собеседованиях. Условие - Напишите программу, которая выводит на экран числа 
от 1 до 100. При этом вместо чисел, кратных трем, программа должна выводить слово Fizz, а вместо чисел, 
кратных пяти — слово Buzz. Если число кратно пятнадцати, то программа должна выводить слово FizzBuzz.
