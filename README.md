## Паттерны
### Singleton
Singleton - порождающий паттерн. Как правило применяется для чтения настроек 
из файла конфигурации, или же в виде класса для подкоючения к базе данных.

### Builder
Builder (Строитель) - это порождающий паттерн проектирования, который позволяет создавать сложные объекты пошагово.
Применить в автотестах можно в качестве файла для создания конфигурации.

### Observer
Паттерн Observer, или Наблюдатель... 

### State
State, или Состояние - это поведенческий паттерн проектирования, который позволяет объектам менять поведение 
в зависимости от своего состояния. Извне создаётся впечатление, что изменился класс объекта.
